# Habit Tracker CLI

A command-line habit tracker built in Python that helps you build consistency through streak tracking.

## Features
- Add and manage daily habits
- Mark habits as complete each day
- Automatic streak tracking
- Duplicate prevention — can't add the same habit twice
- Persistent data storage using JSON
- Can't mark the same habit done twice in one day

## How to Run

```bash
python3 habit_tracker.py
```
